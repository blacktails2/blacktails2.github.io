  • What’s ShiftA?
    ShiftA = Shift + Advance
    ShiftA “シフタ”は、“Shift”=変わる、変える と、“A”=Advance,前進する でできた造語で、 AにはAdvanceの他にも様々な意味が入ります。 そして、ShiftAは自分の作品公開を目的としたブランドです。 現在は、本サイトのShiftA Pagesと、 雑誌を模したポートフォリオShiftA Magazineで展開しています。 ShiftA Pagesでは、Workshiftによる作品紹介、 PortfolioではShiftA Magazineを閲覧できます。
  • About Workshift
    Workshift = Works « Selection
    Workshiftは、「テーマごとに作品を10個選び、公開する」システムです。 フォント、カラー、タイプ、様々なテーマで季節ごとに作品を選んでいき、 ShiftA Pagesのトップページにて紹介しています。 また、過去のWorkshiftも見ることができます。
  • Profile

    Masaki Ando

    安藤 真生

    1998年岐阜県生まれ、デザイナー。デザインの内容はグラフィック、エディトリアル、フロントエンド、UI/UXなど、平面系において多岐にわたる。 サレジオ高専デザイン学科5年に所属し、書体や組版、カリグラフィに興味。

  • About Portfolio
    ShiftA Magazine = Portfolio + Magazine
    Portfolio:ShiftA Magazineは、ライフスタイル誌を模して発行しているポートフォリオです。 2017年3月に第1号「将来を見つめて。」を始め、不定期で刊行しています。 ShiftA PagesのPortfolioではShiftA Magazine以外にも、 過去に作成したポートフォリオを公開しています。
