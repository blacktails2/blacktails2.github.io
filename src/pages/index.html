What’s ShiftA?

ShiftA = Shift + Advance

ShiftAは、私 安藤真生のポートフォリオ・プロジェクトです。自分の制作物の公開を目的に、このホームページ“ShiftA Pages”と雑誌のようなポートフォリオ“ShiftA Magazine”で作品公開を行なっています。
WorkshiftはShiftA Pagesの作品公開システムで、半年に一度、テーマを決めてそれに沿った課題5つを紹介します。Portfolioのページでは、ShiftA Magazineはもちろん、大学進学や様々な目的で作ったポートフォリオを公開しています。

Profile

Workshift 02 / Autumn 2017

Theme: Type

Workshift、2つ目のテーマはType = 文字。将来目指している書体デザイナー。書体は作品の特徴を引き出し、スムーズに情報を伝えてくれます。書体にこだわった課題、5つを紹介。

1. RUSH of Homeworks

2017, by Adobe Illustrator

カードゲームを制作する課題。デザイン学科生になりきり、課題を進めつつ他プレイヤーの邪魔をして課題の進行を遅らせたり、逆に自分の課題を早く進めたりして最終的に評価点を競うドイツゲームです。様々なツールを作成し、フォントはNeue Kabel、タイプラボ、ニューロダンを使用。

  1. Playing
  2. Overview
  3. Cards

2. RC Swow Serif

2017, by Glyphs

サレジオ高専の卒業研究で制作している欧文本文ローマン体。Variable Fontという技術を利用し、自在にセリフをユーザーが調節できる書体を目指しています。

  1. Plastic Jelly Style
  2. Glass Style
  3. Movie Logo Style
  4. Color Metal Style
  5. Text on Text
  6. TRON
  7. Neon Style
  8. I use this now

3. Yumejuya

2016, by Adobe Illustrator

夏目漱石の「夢十夜」特に第一夜の演劇を想定して、ポスターを制作する課題。クラシカルかつモダンなレイアウトと書体選びを心がけました。書体はBaskerville, 毎日新聞明朝をメインに秀英書体やFrutigerなどを使用。

4. A/DO

2015, by Adobe Illustrator

文房具ブランドのコーポレート・アイデンティティを作る課題。シンプルかつ長く使える文房具のブランドを、Helvetica Neueを使って表しました。ツールも様々なものを提案。

5. Type and Illust

2013, by Paints

1年生の基礎課題の一つとして、文字同士や文字とイラストを組み合わせる絵の具の課題。このころは、Helvetica Neueやフリーフォントを中心に扱っていました。

  1. Jackal / October, 2014
  2. Jackal R / January, 2015
  3. Jackal NEXT / December, 2015
